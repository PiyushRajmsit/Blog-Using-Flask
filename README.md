A WEB BLOG BASED ON FLASK, WITH FOLLOWING FEATURES:
1. Create a post
2. Delete a post
3. Edit a post
4. Login
5. Register

STEPS TO INSTALL FOR LINUX:

1.Create a Virtual Environment and activate it
python3 -m venv venv
. venv/bin/activate

2.Install Flask
pip install -e


STEPS TO RUN:
export FLASK_APP=flaskr
export FLASK_ENV=development
flask run
